# 쫄지마

개인용 생존 금융 대시보드 웹앱입니다. 은행 연동이나 API 없이, 직접 입력한 자산/수입/고정비/지출을 기준으로 “아직 얼마나 버틸 수 있는지”를 보여줍니다.

## 1단계 MVP 기능

- 대시보드
- 현재 자산 입력
- 수입 모드 선택
- 고정비 입력
- 지출 기록
- 자동 계산
- 브라우저 localStorage 저장
- 백업 JSON 다운로드/업로드
- 전체 초기화
- 간단한 카테고리 분석

## 데이터 저장 방식

이 앱은 은행 연동이 없고, API도 사용하지 않으며, 서버 파일에 개인 데이터를 저장하지 않습니다.

입력한 데이터는 사용자의 브라우저 `localStorage`에 저장됩니다. 같은 Streamlit URL에 다른 사람이 접속하더라도, 내 브라우저에 저장된 데이터가 다른 사람 브라우저에 보이지 않도록 서버 JSON 저장 방식을 사용하지 않습니다.

주의할 점:

- 브라우저 캐시/사이트 데이터/localStorage를 삭제하면 쫄지마 데이터도 지워질 수 있습니다.
- 핸드폰 브라우저에서 입력한 데이터는 같은 핸드폰의 같은 브라우저에서 다시 접속할 때 유지됩니다.
- 다른 기기나 다른 브라우저에서는 데이터가 자동 공유되지 않습니다.
- 안전하게 쓰려면 `데이터 관리` 탭에서 주기적으로 JSON 백업을 다운로드하세요.

## 백업과 복원

- `백업 JSON 다운로드`: 현재 브라우저에 저장된 데이터를 JSON 파일로 저장합니다.
- `백업 데이터로 복원`: 다운로드해둔 JSON 파일을 다시 업로드해 현재 브라우저에 복원합니다.
- `전체 초기화`: 현재 브라우저의 쫄지마 localStorage 데이터를 삭제합니다.

## 계산 기준

- 총 자산: 현금, 통장 잔고, 비상금, 받을 돈, 기타 자산 합계
- 이번 달 예상 수입: 선택한 수입 모드 또는 직접 입력값
- 이번 달 변동 지출: 현재 월에 기록된 지출 합계
- 이번 달 남은 돈: 예상 수입 - 고정비 - 변동 지출
- 생존 가능 기간: 총 자산 / 월 순소진액
- 월 순소진액: 고정비 + 이번 달 변동 지출 - 예상 수입

월 순소진액이 0 이하이면 현재 입력값 기준으로는 자산이 줄지 않는 상태로 표시합니다.
